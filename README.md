# sorting_algs

Usage: sort [sorting methods] [filling method] min:[max]:[step] [other options]

Sorting methods:
-i      Insertion Sort \
-s      Selection Sort \
-b      Bubble Sort \
-h      Heap Sort \
-q      Quick Sort
-a      Quick Sort Analysis
-Q      Quick Sort 2
-c      Counting Sort
-S      Shell Sort
-2      Shell Sort 2
-3      Shell Sort 3
-m      Merge Sort

Array filling methods:
-I      increasing
-D      decreasing
-R      random
-C      constant
-V      decreasing and then increasing
-X      increasing and then decreasing

Other options:
 --help     display help
     -n     no file output
     -v     verbose
     -A n   average of n
     -r m   max value that the RNG will output
